# Friend Call Page Redesign - Professional Layout

## Problem
The friend call page (CallModal) had a basic design that didn't match the professional Stranger Chat layout. It needed better responsive design and visual consistency.

## Solution Applied

### Redesigned with Stranger Chat Professional Layout

**Key Improvements:**

1. **Fixed Viewport Layout**
   - Changed to `fixed inset-0` for exact viewport fit
   - No scrolling required
   - All controls visible immediately

2. **Full-Screen Video Experience**
   - Remote video fills entire background
   - Picture-in-picture local video (top-right corner)
   - Professional video layout like Zoom/FaceTime

3. **Enhanced Top Bar**
   - Partner avatar and name
   - Live call duration with animated indicator
   - Clean close button

4. **Picture-in-Picture Local Video**
   - Positioned at top-right
   - Rounded corners with border
   - Shows "You" label
   - Camera-off state shows avatar

5. **Audio Call Special View**
   - Beautiful gradient background
   - Large partner avatar with pulse animation
   - Centered layout
   - Clear call duration display

6. **Professional Control Bar**
   - Fixed at bottom
   - Circular buttons with icons
   - Mute/Unmute microphone
   - Camera on/off (video calls only)
   - Large red end call button
   - Volume control (desktop)

7. **Better Status Screens**
   - "Calling..." state with spinner
   - "Connecting..." state
   - Clean, centered layout
